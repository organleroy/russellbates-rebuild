Home Curator (local-only)

This tool is local-only. It does NOT affect the live site until you export and paste the result into: src/_data/home.json.

Workflow:
1) Paste the entire JSON array from src/content/projects.json into the left box.
2) Click Load featured_home only.
3) Drag rows to reorder the “flow”.
4) Click Export. It will export ONLY the first 39 items.
5) Paste the exported JSON into src/_data/home.json.
